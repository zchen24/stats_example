Introduction

Statistics Unplugged is a fairly good introduction book to statistics. To learn how to implement these procedures in MATLAB and how to call MATLAB build-in functions directly, I wrote a couple of scripts organized based on the book chapters. If possible, I first show how to approach these problems using built-in MATLAB functions, followed by a manual way of computing the same procedure. Some chapters barely contain anything new, thus consist only a few solution to Questions. These html pages are generated using MATLAB publish functionality.
