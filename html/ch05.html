Standard Error of the Mean Q1

Question 1

clc; clear;

mu = 24.12;
sigma = 4;
n = 100;

% a
m = mu;
% b
mstd = sigma / sqrt(n);


% print answer
disp(['mean = ', num2str(m), '  std err = ', num2str(mstd)]);

mean = 24.12  std err = 0.4

Question 3
